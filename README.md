# MNIST DATASET - NEURAL NETWORK IN C

This is a simple Neural Network made with C.

## How to compile 

```bash
clang main.c -o main -Wall -Wextra  
clang -std=c11 -Wall -Wextra -pedantic main.c ./src/*.c -lm
```

## How to run

```bash
./main
```

# MNIST Documentation

The dataset used in this project is the [MNIST Database of handwritten digits](https://github.com/sunsided/mnist).
