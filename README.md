# MNIST DATASET - NEURAL NETOWORK IN C

This is a simple MLP Neural Network made with C.

## How to compile 

```bash
clang  main.c -o main -Wall -Wextra  
```

## How to run

```bash
./main
```

# MNIST Documentation

The dataset used in this project is the [MNIST Database of handwritten digits](https://github.com/sunsided/mnist).
